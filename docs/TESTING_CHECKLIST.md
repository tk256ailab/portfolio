## トラブルシューティング

問題がある場合は `TROUBLESHOOTING.md` を参照してください。

### よくある問題

#### ページが真っ白
- ブラウザのコンソールを確認 (F12)
- サーバーが起動しているか確認
- パスが正しいか確認

#### ナビゲーションが動作しない
- components/header.html が読み込まれているか確認
- assets/js/components.js にエラーがないか確認

#### スタイルが適用されない
- CSSファイルが読み込まれているか確認 (Network タブ)
- パスが正しいか確認

#### 画像が表示されない
- 画像ファイルが存在するか確認
- パスが正しいか確認 (index.html: `assets/images/...`, pages/*.html: `../assets/images/...`)

## 完了確認

すべてのチェックが完了したら、GitHub Pagesへのデプロイ準備完了です！

```bash
# Git リポジトリの初期化
git init
git add .
git commit -m "Refactor: Modularize codebase and improve structure"

# GitHub にプッシュ
git remote add origin https://github.com/[username]/portfolio.git
git branch -M main
git push -u origin main
```

その後、GitHub Settings > Pages で設定を行ってください。
